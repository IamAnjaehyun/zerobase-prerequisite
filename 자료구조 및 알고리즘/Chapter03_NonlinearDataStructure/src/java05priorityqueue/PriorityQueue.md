> 우선순위 큐 (Priority Queue)

- 우선순위가 높은 데이터가 먼저 나옴 ( =/= FIFO)
    - 모든 데이터에 우선순위가 있음
    - Dequeue시, 우선 순위가 높은 순으로 나감
    - 우선 순위가 같은 경우는 FIFO

- 우선순위 큐 - enqueue, dequeue
    - 최소 힙 및 최대 힙의 삽입 삭제와 같음

> 우선순위 큐 - 구현

- 배열
- 연결 리스트
- 힙

|             |    enqueue    |  dequeue  |
|:-----------:|:-------------:|:---------:|
|   정렬된 배열    |     O(N)      |   O(1)    | 
|  정렬된 연결리스트  |     O(N)      |   O(1)    | 
|      힙      |    O(logN)    |  O(logN)  |
